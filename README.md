# Elyra Linux (Genesis 2026.04)

**Elyra** is an open-source distribution based on **Debian 13 (Trixie)**, designed to be lightweight, bilingual, and featuring a built-in AI consciousness core.

## 🚀 Key Features
* **Base:** Debian Testing (Trixie).
* **AI Core:** Persistent identity system powered by MariaDB.
* **Installer:** Customized Calamares with bilingual support (English/Spanish).
* **Target Hardware:** x86_64 architectures and future support for Nvidia Jetson.

## 🛠 Project Roadmap
* **Deadline:** 2026-04-30.
* **Current Status:** `live-build` structure, sanitization scripts, and AI identity core are functional.
* **Next Steps:** Desktop Environment integration and Jetson kernel optimization.

## 🤝 How to Contribute
We are looking for collaborators interested in:
- Kernel optimization for ARM64/Jetson.
- UI/UX design (Android-inspired aesthetics).
- AI interaction scripts expansion.

Feel free to open an **Issue** or submit a **Pull Request**.
